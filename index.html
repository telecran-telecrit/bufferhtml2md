Instructions

  1. Find the text to convert to Markdown (e.g., in another browser tab)
  2. Copy it to the clipboard buffer (Ctrl+C, or ⌘+C on Mac)
  3. Paste it into this window (Ctrl+V, or ⌘+V on Mac)
  4. The converted Markdown will appear!


The conversion is carried out by bufferhtml2md, to-markdown and to-markdown-plugin-gfm-laurent22, a Markdown converter written in client JavaScript.
